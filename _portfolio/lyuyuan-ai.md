## Lyuyuan AI

Lyuyuan_AI is the public historical prototype for a Green Garden High School narrative RPG. It explores LLM-driven character dialogue, relationship state, event tracking, and lightweight save/load flows.

### Highlights
- Multiple selectable AI characters with distinct persona prompts.
- A two-channel LLM response pattern: character reply plus internal analysis JSON.
- Relationship and emotion state updates through events.
- Event bus architecture for decoupled logging, statistics, achievements, and lifecycle events.
- Lightweight JSON save slots and a Flask-based web UI.

### Stack
- Python
- Flask
- DeepSeek/OpenAI-compatible LLM providers
- Event-driven architecture
- HTML/Bootstrap/jQuery

## Lyuyuan AI

Lyuyuan_AI 是“绿园中学物语”的公开历史原型仓库，探索 LLM 驱动的多角色对话、关系状态、事件追踪和轻量存档流程。

### 亮点
- 多个可选 AI 角色，每个角色拥有独立 persona prompt。
- 一次 LLM 请求同时产出角色回复和内部分析 JSON。
- 通过事件更新好感度、关系阶段和情绪状态。
- 事件总线架构支持日志、统计、成就和游戏生命周期事件解耦。
- 使用 JSON 存档槽位和 Flask Web UI 实现轻量交互。

### 技术栈
- Python
- Flask
- DeepSeek/OpenAI-compatible LLM providers
- Event-driven architecture
- HTML/Bootstrap/jQuery
