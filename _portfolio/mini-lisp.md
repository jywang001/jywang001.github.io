## Mini Lisp Interpreter

Mini-Lisp is a C++ interpreter built for Peking University's Software Design Practice course. It focuses on clear tokenizer/parser design, evaluation environments, core Lisp forms, a REPL, script execution, and a useful subset of builtins.

### Features
- REPL and script execution.
- Numeric, boolean, string, symbol, pair/list, and empty-list values.
- Special forms including `define`, `lambda`, `if`, `begin`, `let`, `cond`, `and`, `or`, `quote`, `quasiquote`, `unquote`, and `define-macro`.
- Builtins for I/O, type predicates, list processing, numeric comparison, arithmetic, higher-order functions, and strings.
- CMake-based build flow.

### Stack
- C++
- CMake
- Interpreter design
- Functional programming

## Mini Lisp 解释器

Mini-Lisp 是为北京大学《软件设计实践》课程练习实现的 C++ 解释器，重点在清晰的词法/语法解析、求值环境、核心 Lisp 形式、REPL、脚本执行和实用内建过程。

### 功能
- 支持 REPL 和脚本执行。
- 支持数字、布尔、字符串、符号、pair/list 和空表等值类型。
- 支持 `define`、`lambda`、`if`、`begin`、`let`、`cond`、`and`、`or`、`quote`、`quasiquote`、`unquote`、`define-macro` 等特殊形式。
- 内建过程覆盖 I/O、类型判断、列表处理、数值比较、算术、高阶函数和字符串操作。
- 使用 CMake 构建。

### 技术栈
- C++
- CMake
- Interpreter design
- Functional programming
